class Lenti_Documentation.lenti_test.TestDataFrameProcessing(methodName='runTest')[source]

Bases: TestCase

This class defines test cases to validate the cleanliness and integrity of DataFrames. It includes tests for column names, rows with missing values, and integrity checks for key columns in DataFrames.

run_tests_for_dataframe(df, df_name, key_columns_dict)[source]

Runs a series of tests on a given DataFrame and returns the results. Tests include column name cleanliness, row null checks, row count, and key column integrity.

Args:

df (pd.DataFrame): The DataFrame to test. df_name (str): Name of the DataFrame being tested. key_columns_dict (dict): Dictionary mapping DataFrame names to key columns for integrity checks.

Returns:

list: A list of tuples containing test case name, DataFrame name, test status, failure reason, and runtime.

test_integrity_check(df, df_name, key_columns_dict)[source]

Test to check the integrity of the DataFrame by ensuring key columns do not contain null values. Key columns are provided through a dictionary where DataFrame names are mapped to the key columns.

df (pd.DataFrame): The DataFrame to test. df_name (str): Name of the DataFrame being tested. key_columns_dict (dict): Dictionary containing DataFrame names as keys and a list of key columns as values.
